Preparation

You will need:

  1. A tablet or smartphone running Android 7+ equipped with a camera (there are separate instructions for older Android versions).
  2. WiFi Internet connection (for the tablet).

Important

The functionality needed for these installation steps, "QR provisioning", is part of Android 7, however there are devices on which it doesn't work.

1. Factory reset

The device needs to be cleared to its factory state so that Homer Player can be installed as a priviliged app. This will allow it to activate the full kiosk mode.

Factory reset deletes all contacts, photos, music and other data from the device.
Make sure you copy all the important data to other devices before proceeding.

  1. Go to Settings → Backup & reset.
  2. Tap Factory data reset to reset the device.
  3. Reset will take a while. When it's finished do not start configuring the device, see the next step.

2. Device setup with Homer Player

When the factory reset process finishes you are presented with the device "Welcome" screen.

  1. Tap the "Welcome" screen 6 times (somewhere not on a button).
  2. Follow the on-screen instructions to connect to the Internet and install additional components.
  3. A camera app for scanning QR codes will show up.
  4. Point the camera at the QR code below to scan it.
    The code instructs the device to install Homer Player and make it a priviliged app so that it can activate the full kiosk mode.
  5. Follow the on-screen instructions to finish installation of Homer Player.

If tapping the "Welcome" screen doesn't have any effect it is likely your device doesn't support "QR provisioning".
Please e-mail me, I may be able to help.

3. Enable the full kiosk mode

Now you can enable and disable the full kiosk mode at will.

  1. Start Homer Player.
  2. Tap 5 times on the screen to enter settings.
  3. Tap Prevent from exiting the application (kiosk mode).
  4. Enable Full mode...
  5. Go back to leave the settings screen.
Now it's impossible to leave the player application (you can revert this by disabling the "Full mode..." option).
